== Description ==

This theme is perfect for any kind of blog of your own choice. Translation and multi-Iingual (multiple languages) feature is here for your convinience. Multiple widget areas allow you set up your blog your own way.

* Front Page Featured Posts		
* Image and Text Logos		
* Background Image		
* Background Color	
* Special Sticky Posts Format with Signature	
* Grid and List Blog Feed Layouts
* Recent Posts Widget With Thumbnails
* Easy to Create Social Media Icon Menus
